# Course Design Plan — Leveraging LLM Agents with Skills (V2)

## Overview
Redesign of the existing "Sử dụng Skills để khai thác sức mạnh LLM Agent" course with four major upgrades:
1. **Rich visual design**: Canvas-design infographics, custom illustrations on every slide
2. **Interactive artifacts**: HTML/React learning tools (skill simulators, interactive flowcharts, decision trees)
3. **Scaffolded learning**: Micro-step guided templates, fill-in-the-blank exercises, checkpoints before free-form practice
4. **Industry-specific scenarios**: HR, Marketing, Finance, Ops case studies replacing generic examples

## Target Learner Summary
- Non-technical business users (office workers, managers, marketing/HR/finance/ops specialists)
- Already use ChatGPT/Claude for basic chat; don't know agents/skills/MCP
- Visual + Kinesthetic learners, 70% hands-on preferred
- 5 sessions × 1 hour, online via Zoom/Teams, Vietnamese

## Discovery Interview Parameters
- **Learners:** Non-technical business users (same as v1)
- **Outcome:** Use + Create skills
- **Duration:** 5 × 1 hour
- **Delivery:** Virtual (Zoom/Teams)
- **Language:** Vietnamese
- **Deliverables:** Full package + interactive HTML artifacts

## V2 Enhancements over V1

| Dimension | V1 (Current) | V2 (Target) |
|-----------|-------------|-------------|
| Slides | PptxGenJS shapes/text only | Canvas-design infographics + rich illustrations |
| Activities | Text worksheets | Scaffolded micro-step templates + interactive HTML tools |
| Scenarios | Generic examples | 4 industry tracks (HR, Marketing, Finance, Ops) |
| Learning aids | Static handouts | Interactive React artifacts (skill builder, decision tree) |
| Assessment | MCQ quizzes | MCQ + interactive self-check tools |
