# Course Design Plan — Prompt Engineering with Claude Code & Cowork

## Overview
Adapt existing "Prompt Engineering cho người dùng doanh nghiệp" course to emphasize Claude Code CLI and cowork collaboration patterns as the primary application tool. Same core prompt engineering pedagogy, recontextualized for terminal-based AI collaboration.

## Key Adaptation Decisions
- **Tool shift**: ChatGPT/Claude web → Claude Code CLI + cowork workflows
- **New concepts**: CLAUDE.md config, slash commands, skills, agent delegation, MCP servers, cowork patterns
- **Retained**: All prompt frameworks (RTT, CO-STAR, few-shot, CoT), scaffolding progression, ADDIE structure
- **Language**: Vietnamese (same as original)
- **Audience**: Same non-technical business users, but now learning CLI-based AI workflow

## Target Learner Summary
- Business users (HR, Marketing, Finance, Ops) exploring Claude Code for productivity
- Prior: familiar with web-based AI chat, new to CLI/terminal-based AI tools
- Learning style: visual + kinesthetic, need immediate before/after results
- Delivery: 5 sessions × 60 min, online via Zoom/Teams

## Discovery Interview Parameters
- **Learners:** Non-technical business users adopting Claude Code
- **Outcome:** Use Claude Code + cowork for daily business tasks
- **Duration:** 5 × 60 min
- **Delivery:** Virtual (Zoom/Teams)
- **Language:** Vietnamese
- **Deliverables:** Full package (slides, guide, handout, activities, assessments)
