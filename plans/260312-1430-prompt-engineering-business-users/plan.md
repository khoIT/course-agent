# Course Design Plan — Prompt Engineering cho người dùng doanh nghiệp

## Overview
5-session (5×60 min) Vietnamese-language course teaching non-technical business users systematic prompt engineering. Companion to the "Leveraging LLM Agents" course — this covers the foundational communication skill that makes agents effective.

## Target Learner Summary
- Non-technical business users (HR, Marketing, Finance, Ops) already using ChatGPT/Claude casually
- Know how to chat with AI but get inconsistent results; don't use frameworks
- Visual + kinesthetic learners; lose interest with too much theory
- Online delivery via Zoom/Teams with breakout rooms

## Discovery Interview Parameters
- **Learners:** Non-technical business users across departments
- **Outcome:** Apply systematic prompt frameworks to daily tasks
- **Duration:** 5 sessions × 60 min
- **Delivery:** Virtual (Zoom/Teams)
- **Language:** Vietnamese
- **Deliverables:** Full package (slides, guide, handout, activities, assessments)
