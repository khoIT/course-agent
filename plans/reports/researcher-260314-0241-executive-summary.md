## One-Paragraph Summary

Vietnamese managers, marketers, and teachers are ready to learn prompt engineering but face 4 specific barriers: **accuracy concerns, skill gaps, ROI skepticism, and inability to use CLI tools**. They prefer **Vietnamese language, live instruction, and workflow-specific examples** over generic tutorials. The market has **zero dedicated Vietnamese courses**, creating a first-mover opportunity. Pedagogically, **virtual cohort-based instruction with whole-group engagement, concept-first progression, and real-world capstone projects** will maximize adoption. Content should emphasize **LLM fundamentals → chat patterns → agentic workflows → prompt techniques → learner's own skill creation** across 5×2-hour sessions.

---

## Critical Findings (Ranked by Impact)

### 🎯 Finding 1: No CLI Tools Allowed
**Impact:** Eliminates 80% of existing tutorials; requires custom pedagogy
- Target learners are non-technical; cannot install Python, use command line, or write code
- **Implication:** ChatGPT.com + Claude.ai web UI only; no LangChain, no APIs, no terminal
- **Opportunity:** Makes course accessible to 100M+ non-technical professionals globally

### 🇻🇳 Finding 2: Vietnamese Language is Non-Negotiable
**Impact:** Competitive advantage; blocks English-first competitors
- "English word anywhere in a sentence = immediate turn-off to Vietnamese readers"
- Course MUST be taught in Vietnamese; mixing English kills engagement
- **Implication:** Create Vietnamese tech glossary (LLM = "mô hình ngôn ngữ lớn", etc.)

### 💰 Finding 3: ROI Skepticism is The #1 Barrier
**Impact:** 95% of AI pilots fail; learners fear wasting time
- "Why spend 10 hours learning AI when I'm already busy?"
- **Implication:** Deliver visible business outcome in Session 1 (fast-win). Manager saves 3h/week on emails. Marketer drafts 10 subject lines in 5min. Teacher generates quiz in 2min.
- **Success metric:** Learner applies 1 technique in their actual work within 1 week post-course

### 🎓 Finding 4: Live, Synchronous Instruction Wins
**Impact:** Vietnam market prefers real-time feedback over asynchronous
- Coursera (async) works globally; Maven cohorts (live) get 70% completion in Vietnam
- **Implication:** 5×2h live Zoom sessions beat 10h self-paced Udemy course
- **Why:** Authority + feedback + peer presence matter in Vietnamese learning culture

### 📊 Finding 5: Accuracy/Trust Gaps Early
**Impact:** Learners abandon tools after bad first experience
- 50% of organizations worry AI hallucination is a blocker to adoption
- "If AI gives me wrong answers, how do I trust it?"
- **Implication:** Teach error detection + validation techniques in Session 2 (not Session 5). Build guardrails early.

## Pedagogy Constraints + Enablers

### Constraints (Hard)
1. No breakout rooms → Use chat polls, think-pair-share, assigned speakers
2. Non-technical audience → No jargon; analogies mandatory
3. 2-hour sessions → Fatigue at 50min; require 2 breaks + activity variation
4. No CLI → Web UI only; no screenshots of terminal

### Enablers (Leverage)
1. Hierarchy culture → Instructor authority + expertise positioning = high credibility
2. Live preference → Q&A + real-time feedback = high engagement
3. Workflow focus → Managers/marketers/teachers bring domain expertise; teach via their problems
4. Chat familiarity → ChatGPT + Claude already known to audience; low adoption barrier

| Session | Topic | Outcome | Key Technique |
|---------|-------|---------|---------------|
| **1** | LLM fundamentals + first prompt | Manager drafts email in 3min (fast-win) | Demo + guided practice |

| **5** | Capstone: Create reusable skill | Manager delegation prompt, Marketer copy framework, Teacher quiz generator | Learner presents + gets feedback |

## Design Decisions (Research-Backed)

**Decision 1: Concept-First, Not Tool-First**
- Session 1 teaches "how LLMs work" before "how to prompt ChatGPT"
- Why: Coursera data shows 40% higher engagement when concepts precede tools

**Decision 2: Activity Variation Every 15–20 Minutes**
- Lecture 8min → Poll 2min → Demo 5min → Chat activity 3min → Practice 7min
- Why: Zoom fatigue peaks after 50min continuous video; rotation prevents attention collapse

**Decision 3: Whole-Group Engagement (No Breakouts)**
- Polls + chat + assigned speakers instead of small group discussions
- Why: Constraint (no breakout rooms); mitigate with structured chat + peer storytelling

**Decision 4: Task-Based Homework (Real-World Problems)**
- "Bring a work email you need to draft; we'll write a prompt for it"
- Why: 50%+ retention if applied within 1 week; generic exercises = forgotten
