# Audience Analysis Report
## Prompt Engineering for Business Users (Vietnam)

**Report Date:** 2026-03-14
**Confidence:** Medium (Global data solid; Vietnam-specific gaps noted)

---

## Critical Findings

### 1. Four Adoption Barriers (Priority Order)
1. **Accuracy/Trust Deficit** — 50% of organizations worry AI accuracy/bias; SMEs demand oversight
2. **Skill Gap + Confidence** — 40% of enterprises lack internal AI expertise; 51% at management level
3. **ROI Skepticism** — 95% of AI pilots fail; <20% scale post-launch; learners fear wasting time
4. **Technical Barriers** — Non-technical users cannot use CLI tools; glossy UI required

### 2. Vietnamese Context (Unique Factors)
- **Digital skills gap:** Only 11% of workforce Industry 4.0–ready; however, 20%+ annual growth + gov mandate for digital ops by June 2025
- **Language:** English words mixed into Vietnamese text = "immediate turn-off." Require full Vietnamese translation; code-switching demotivating
- **Workplace culture:** Hierarchical, rote-learning preferred, authority deference strong; critical thinking undervalued
- **AI adoption unknown:** No published Vietnam ChatGPT/Claude split. Assume ChatGPT-first (global dominance); Claude as secondary option

### 3. Market Reality (Global)
- ChatGPT: 68% market share, 2.8B MAU (but down from 87% year-ago) → vulnerable to alternatives
- Claude: Fastest growth late 2025; 70% Fortune 100; 46% for work (credibility rising)
- Google Gemini: Explosive (5.4% → 18.2% in 12 months) → consider as co-tool

### 4. Tool Implications (Direct to Course Design)
- **NO CLI TOOLS** — learners are non-technical; Python, LangChain, bash impossible
- ChatGPT web UI + Claude web UI only
- Avoid code examples; teach through iterative web-based prompting
- Add error-detection + guardrail techniques early to build confidence

---

## Design Implications by Learner Type

| Role | Pain Point | Course Action |
|------|-----------|---------------|
| **Managers** | ROI skeptical; time-poor | Session 1: 15-min case study showing $impact (e.g., email drafting saves 3h/week) |
| **Marketers** | Don't know how to "ask" AI; accuracy fear | Teach copy-specific templates; error-detection workflow |
| **Teachers** | Plagiarism risk; tech intimidation | Focus exam rubric + lesson design prompts; ethical framing; no CLI |
| **All (VN)** | English overwhelming | Full Vietnamese; glossary; hierarchical instructor model |
