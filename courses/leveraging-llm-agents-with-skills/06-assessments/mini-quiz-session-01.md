# Mini-Quiz — Buổi 1: Agent & Skill 101

> 3 câu trắc nghiệm nhanh | Dùng Mentimeter, Zoom Poll, hoặc chat
> Thời điểm: Phút 55-60 của Buổi 1

---

## Câu 1: Khái niệm cốt lõi

**Trong analogy "nhân viên mới", Skill tương ứng với gì?**

- A. Phần mềm công ty (Word, Excel...)
- B. Sổ tay quy trình (SOP) ✅
- C. Bàn làm việc và máy tính
- D. Tên và chức danh nhân viên

> **Đáp án: B** — Skill = SOP hướng dẫn agent biết phải làm gì và làm như thế nào. Tool = phần mềm, Agent = nhân viên.

---

## Câu 2: Phân biệt Prompt vs Agent + Skill

**Điểm khác biệt LỚN NHẤT giữa prompt đơn lẻ và agent có skill?**

- A. Agent trả lời nhanh hơn
- B. Agent cho kết quả nhất quán và có quy trình ✅
- C. Agent dùng được nhiều ngôn ngữ hơn
- D. Agent miễn phí còn prompt thì không

> **Đáp án: B** — Skill giúp agent có "quy trình chuẩn" → kết quả nhất quán mỗi lần, thay vì phụ thuộc vào cách viết prompt từng lần.

---

## Câu 3: Ứng dụng

**Công việc nào sau đây PHẢI có skill riêng thay vì dùng prompt đơn?**

- A. Hỏi "thủ đô Việt Nam là gì?"
- B. Nhờ AI dịch 1 câu sang tiếng Anh
- C. Tạo báo cáo tuần theo đúng template công ty ✅
- D. Hỏi ý nghĩa một từ tiếng Anh

> **Đáp án: C** — Báo cáo tuần cần đúng format, đúng mục, đúng style → cần skill chứa template và quy trình. Các việc còn lại đơn giản, prompt thường đủ.
