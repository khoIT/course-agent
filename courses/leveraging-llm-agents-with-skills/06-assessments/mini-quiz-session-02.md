# Mini-Quiz — Buổi 2: Sử dụng Skills có sẵn

> 3 câu trắc nghiệm | Dùng cho ôn tập đầu Buổi 3 hoặc cuối Buổi 2
> Mục tiêu: Kiểm tra khả năng chọn đúng skill cho đúng việc

---

## Câu 1: Chọn Skill đúng

**Bạn cần tạo file Word tóm tắt cuộc họp. Skill nào phù hợp nhất?**

- A. pptx
- B. xlsx
- C. docx ✅
- D. pdf

> **Đáp án: C** — docx tạo file Word (.docx). pptx cho slide, xlsx cho bảng tính, pdf cho đọc/tạo PDF.

---

## Câu 2: Chuỗi Skills

**Bạn cần: đọc dữ liệu từ PDF → phân tích số liệu → tạo slide trình bày. Thứ tự skill đúng là?**

- A. pptx → xlsx → pdf
- B. pdf → xlsx → pptx ✅
- C. docx → pdf → pptx
- D. xlsx → pdf → docx

> **Đáp án: B** — pdf (đọc/trích xuất dữ liệu) → xlsx (phân tích bảng tính) → pptx (tạo slide trình bày).

---

## Câu 3: Tình huống thực tế

**Sếp gửi email: "Tạo infographic tóm tắt KPI quý này từ file Excel đính kèm." Bạn cần dùng skill nào?**

- A. Chỉ cần xlsx
- B. xlsx + canvas-design ✅
- C. Chỉ cần canvas-design
- D. docx + pptx

> **Đáp án: B** — xlsx để đọc/phân tích dữ liệu KPI, canvas-design để tạo infographic trực quan.
