## Phần 1: Tổng Quan Khóa Học

### 1.1 Thông tin khóa học cơ bản

| Mục | Chi tiết |
|-----|---------|
| **Thời lượng** | 5 buổi × 60 phút |
| **Hình thức** | Online qua Zoom/Teams |
| **Ngôn ngữ** | Tiếng Việt |

### 1.2 Mục tiêu khóa học (10 Learning Objectives)

1. Giải thích sự khác biệt giữa prompt đơn lẻ và LLM agent có skill
2. Xác định công việc lặp lại có thể giao cho agent
3. Kích hoạt và sử dụng 3+ skills có sẵn trong Cowork
4. Lựa chọn skill phù hợp bằng decision tree tương tác
5. Tạo file SKILL.md hoàn chỉnh bằng interactive Skill Builder
6. Tổ chức thư mục skill với references/ và cấu trúc chuẩn
7. Thiết kế skill phức tạp có references, multi-step workflow
8. So sánh Cowork vs Claude Code bằng decision tree
9. Đánh giá chất lượng skill dựa trên rubric tương tác
10. Xây dựng kế hoạch triển khai skills cho team

### 1.3 Chân dung học viên

- **Nhóm chính**: Người dùng phi kỹ thuật muốn tận dụng AI agents để tự động hóa công việc
- **Trạng thái**: Đã dùng ChatGPT/Claude nhưng chưa hiểu "agent", "skill"

### 1.4 Yêu cầu kỹ thuật

- Zoom hoặc Microsoft Teams
- Truy cập Cowork platform
- Trình duyệt web cập nhật
- Tài khoản Claude AI

## Phần 4: Phụ Lục

### 4.1 Glossary

| Thuật ngữ | Định nghĩa |
|-----------|-----------|
| **Agent** | Trợ lý AI tự động theo quy trình (skill). System-based, không chat-based. |
| **Skill** | Tài liệu (SKILL.md) định nghĩa tên, mô tả, instructions. Tương tự SOP. |
| **Tool** | Phần mềm (email, Slack, database) mà agent có thể dùng. |
| **Cowork** | Platform có skills sẵn (pptx, docx, pdf, xlsx, canvas). Dễ beginner. |
| **Claude Code** | CLI cho advanced users & custom workflows trên máy local. |
| **References** | Thư mục tài liệu hỗ trợ skill. Giúp agent có kiến thức chuyên sâu. |
| **SKILL.md** | File markdown: YAML frontmatter + body instructions. |
| **Workflow** | Chuỗi skills chạy liên tiếp (pdf → xlsx → docx → pptx). |

### 4.2 Xử lý tình huống thường gặp

**Vấn đề**: Nhầm agent = mô hình AI, skill = prompt

**Giải pháp**: Dùng analogy GPS: "Lần đầu hỏi đường từng lúc, lần này input địa chỉ một lần, GPS tự điều hướng."
