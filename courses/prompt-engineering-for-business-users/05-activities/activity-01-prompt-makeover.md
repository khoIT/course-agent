# Activity 01: Prompt Makeover

> **Session:** 1 | **Duration:** 15 min | **Type:** Group Practice | **Objective:** LO1, LO2

## Overview
Học viên mang prompt THẬT từ công việc, phân tích lỗi, viết lại theo công thức Role + Context + Task + Format, và test output.

## REAL Criteria Check
- **R**elevant: Dùng prompt thật từ công việc — không phải ví dụ giả ✓
- **E**ngaging: Before/After comparison tạo "wow moment" ✓
- **A**ctive: Viết lại + test trên AI tool → produce real output ✓
- **L**earner-centered: Mỗi người dùng task riêng, context riêng ✓

## Materials Needed
- Worksheet (dưới đây)
- ChatGPT hoặc Claude access (mỗi học viên)
- Breakout rooms (3-4 người)

## Procedure

### Setup (2 min)
1. Giảng viên share worksheet link trong chat
2. "Mở 1 prompt bạn đã dùng gần đây — từ chat history hoặc từ homework. Paste vào Bước 1."
3. Send to breakout rooms (3-4 người)

### During Activity — Facilitator Micro-Actions (13 min)

**Roaming schedule:**
- 0:00–0:03: Visit Room 1 → Room 2 (quietest rooms first). Spend 90 sec each.
- 0:03–0:06: Visit Room 3 → Room 4. Focus on: have they started step 2?
- 0:06–0:09: Revisit any room that was stuck. Help with step 3 (rewrite).
- 0:09–0:12: Quick check all rooms — who's testing on AI?
- 0:12–0:13: Return to main room, prepare debrief.

**What to listen for:**
- "Tôi không biết prompt nào dùng" → "Mở chat history ChatGPT, scroll xuống — bất kỳ prompt nào."
- "Prompt tôi OK rồi, không cần sửa" → "Thử thêm Role và Format — test xem output có khác không?"
- "AI cho ra kết quả giống nhau" → "Check: bạn đã thay đổi đủ chưa? Thêm specific context về company/department."

**Intervention triggers:**
- Nhóm chưa bắt đầu sau 2 min → "Bắt đầu từ Bước 1: paste prompt gần nhất. Đừng nghĩ nhiều — paste trước, phân tích sau."
- Nhóm stuck ở Bước 2 (không biết thiếu component nào) → "Hỏi: prompt này có nói AI đóng vai gì không? Nếu không → thiếu Role."
- Nhóm skip step 4 (không test) → "PHẢI test. Paste prompt mới vào ChatGPT/Claude — so sánh output cũ vs mới."

**Early finisher instructions:**
- "Xong rồi? Challenge: viết lại prompt CÙNG task nhưng cho audience khác (VD: email cho khách hàng → email cho sếp). Thay đổi gì?"

**Time-check announcements:**
- 0:05: "Các bạn đã xong Bước 2 chưa? Nếu chưa, tập trung vào component QUAN TRỌNG NHẤT thiếu."
- 0:10: "Còn 5 phút — hãy test prompt mới trên ChatGPT/Claude ngay. So sánh output."
- 0:13: "Quay lại main room trong 30 giây. Chuẩn bị share: prompt trước vs. sau."

## Worksheet

### Bước 1: Prompt gốc
Paste prompt bạn đã dùng gần đây:
```
[Paste prompt here]
```

### Bước 2: Phân tích — thiếu component nào?
| Component | Có? | Ghi chú |
|-----------|-----|---------|
| Role (AI đóng vai gì?) | ☐ Yes / ☐ No | |
| Context (thông tin nền?) | ☐ Yes / ☐ No | |
| Task (việc cụ thể?) | ☐ Yes / ☐ No | |
| Format (output trông thế nào?) | ☐ Yes / ☐ No | |

### Bước 3: Viết lại
```
Role: [AI đóng vai...]
Context: [Bối cảnh cụ thể...]
Task: [Việc cần làm...]
Format: [Output mong muốn...]
```

### Bước 4: Test & So sánh
| | Output cũ | Output mới |
|---|-----------|-----------|
| Accuracy | /5 | /5 |
| Relevance | /5 | /5 |
| Useability | /5 | /5 |

### Bước 5: Key Learning
Thay đổi nào tạo khác biệt lớn nhất? _______________

## Debrief Questions
1. Component nào thường bị thiếu nhất trong nhóm?
2. Thay đổi nhỏ nào tạo khác biệt lớn nhất trong output?
3. Prompt nào trong nhóm có cải thiện ấn tượng nhất?
